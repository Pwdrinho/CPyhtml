## 4. Estruturas de Dados: Listas e Tuplas

Listas s√£o cole√ß√µes ordenadas e **mut√°veis** de elementos. Podemos armazenar diferentes tipos de dados em uma lista.

#### Criando e Manipulando Listas

## 5. Dicion√°rios
Dicion√°rios s√£o cole√ß√µes de pares `chave: valor`, usados para armazenar dados de forma estruturada.
```python

# Acessando valores
print(aluno["nome"])  # Sa√≠da: Ana

# Adicionando e alterando valores
aluno["idade"] = 23  # Modificando um valor
aluno["nota"] = 9.5  # Adicionando um novo par

# Removendo um elemento
del aluno["curso"]

# Percorrendo um dicion√°rio
for chave, valor in aluno.items():
    print(f"{chave}: {valor}")
```

## 8. Manipula√ß√£o de Arquivos

### Leitura e Escrita em Arquivos
#### `open` 
A fun√ß√£o `open()` √© utilizada para a abertura dos arquivos.
#### `write`
A fun√ß√£o `write()` √© utilizada para gravar informa√ß√µes em um arquivo existente.
#### `close`
A fun√ß√£o `close()` √© utilizada para encerrar o arquivo ap√≥s sua utiliza√ß√£o.
#### `read`
A fun√ß√£o `read()` realiza a leitura de todo o conte√∫do do arquivo
- Par√¢metros:
    - `‚Äòr‚Äô` que representa que o arquivo est√° sendo aberto em **modo leitura**.

    - `r+` Abre o arquivo para **leitura e escrita**. O arquivo deve j√° existir. A escrita come√ßa a partir da primeira linha e, caso exista algo escrito no arquivo, as linhas ser√£o reescritas, conforme formos escrevendo.
    - `w` Abre o arquivo **somente para escrita**, no in√≠cio do arquivo. Apagar√° o conte√∫do do arquivo se ele j√° existir. Criar√° um arquivo novo se n√£o existir.
    - `w+` Abre o arquivo para **escrita e leitura**, apagando o conte√∫do preexistente.
    - `a` Abre o arquivo para **escrita no final do arquivo**. N√£o apaga o conte√∫do preexistente.
    - `a+` Abre o arquivo para **escrita no final do arquivo e leitura**.

---
```python

# Escrevendo em um arquivo

    f.write("Ol√°, mundo!\n")
    f.close()

# Lendo um arquivo

    conteudo = f.read()

```

# Exemplo Integrado
O c√≥digo abaixo combina listas, dicion√°rios, manipula√ß√£o de strings e fun√ß√µes para criar um sistema simples de cadastro de alunos.

```python
def cadastrar_aluno(nome, idade, curso):
    aluno = {"nome": nome, "idade": idade, "curso": curso}
    return aluno

# Lista para armazenar os alunos
alunos = []

# Adicionando alunos
alunos.append(cadastrar_aluno("Alice", 22, "Matem√°tica"))
alunos.append(cadastrar_aluno("Bob", 20, "F√≠sica"))

# Exibindo informa√ß√µes formatadas
def exibir_alunos(lista):
    for aluno in lista:
        print(f"Nome: {aluno['nome']} - Idade: {aluno['idade']} - Curso: {aluno['curso']}")

exibir_alunos(alunos)

# Salvando em JSON
import json
with open("alunos.json", "w") as f:
    json.dump(alunos, f)

# Lendo o JSON
def carregar_alunos():
    with open("alunos.json", "r") as f:
        return json.load(f)

print(carregar_alunos())
```
